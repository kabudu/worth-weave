# Support

Worthweave is community-supported software and does not currently provide a service-level agreement.

## Get help

1. Read the [README](README.md), [import documentation](docs/data-imports.md), and [release documentation](docs/release.md).
2. Search existing GitHub issues.
3. Open a support issue with the Worthweave version, macOS version, broker, account type, expected behavior, and sanitized diagnostic details.

Never attach broker exports, portfolio screenshots containing personal values, database files, encrypted backups, credentials, account identifiers, or signing material. Reproduce data problems with synthetic files.

Feature proposals should use the feature-request template and explain the reporting decision they enable. Security reports must follow [SECURITY.md](SECURITY.md).

Worthweave provides portfolio analysis, not financial, tax, or investment advice.
